# Tool categories

Load a category with `load_tool_guide` before using its specialized tools.
Core tools (`load_tool_guide`, memory helpers) are always available.
After loading, the API receives native function schemas — these notes are tips only.

| Category | When to load |
|---|---|
| `memory` | Update profile / soul / long-term notes |
| `files` | Local files and folders |
| `system` | Shell, processes, disk, clock |
| `web` | Search or fetch pages |
| `apps` | Open apps and paths |
| `code` | Repo inspect, git, patches |
| `reminders` | One-shot / daily reminders |
| `media` | Clipboard, screenshots, media list |

Prefer one category at a time.
